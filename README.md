# CFSSReview

1) Go to [ANES website](https://electionstudies.org/) and download the 2012 time series study.
2) Import it into your .Rmd as the dataframe ANES12.
3) Make sure you have the following packages:

```

library("tidyverse")
library("psych")
library("lavaan")
library("semPlot")
library("gridExtra")
library("grid")
library("lattice")
library("xlsx")
library("readstata13")
library("ggpubr")
```
